Free to Use - No Registration Required

Betta is a free, open source web application created by DabApps. You don't need to register, like us on Facebook, or any of that.

Just jump straight into the app and start creating!

Quick and Easy to Use

Betta takes Bootstrap's variables and gives you an easy to use interface, allowing you to quickly create unique styles without writing any less / css yourself.

With our instant previews, minimal interface, quick search feature & color picker, you'll have a shiny new design in no time.

Import & Export

Import variables from an existing project, or export variables created with Betta.

There are options for importing & exporting variables that allow you to easily merge styles from multiple projects and only export the things you care about.
